CDE-Nuance Communications

Project Context

CDE One is a medical coding tool that clinicians use for insurance purposes. For CDE One, the search bar they used for finding medical codes needed to be redesigned

Problem

Nuance CDE One search filter was too clunky and it needed to be redesigned

Solution

Create a new, intuitive search filter for users.

Redesign Process

For this project, I had to help redesign the search filter and create a low fidelity mockup of the redesigned filter. With the help of my supervisor, I was able to create the filters for the system. I cannot show the entire system. However, I can show some of the changes that were made. First, we combined two tab systems into one tab by having users click through the list next to the search bar to help reduce the number of clicks.

Before-Two Tabs

After- Merged Tabs (Tab is on the left)

Instructions were also added to allow users to know the character limit for the tool. This helped users save time and narrow down on their decisions for entering search terms.

Before- No Instructions

After- Instructions Added

The search filters also had sort filters that would results in alphabetical order and reverse alphabetical order to filter results faster.

After- Search Filter

The results section was added to allow users to select different medical conditions to narrow down search results.

Limitations

For the CDE One project, I had no choice but to rely on my supervisor’s feedback to understand the user’s needs. I also struggled with figuring out how to make the changes

Conclusion

This particular internship threw me for a curveball as I had to learn how to create proper user interfaces from scratch. Although, I learned how to create user interfaces in class, this internship made me realize I needed to go back to the basics.

But as I worked with the product, I learned about the potential and uses for it outside the medical industry. It made me realize that I wanted to continue to work with this type of technology in the future. This internship also made me realize that I want to continue to work in the medical field going forward.
